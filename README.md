# Login Page

A modern and responsive login page built with HTML, CSS, and JavaScript.

## ✨ Features

* Clean and modern UI
* Responsive design
* Client-side form validation
* Smooth user experience
* Easy to customize

## 🛠 Built With

* HTML5
* CSS3
* JavaScript

This project is a simple login page designed to demonstrate front-end development concepts, including responsive layouts, styling techniques, and interactive form validation using JavaScript.

## 🚀 Getting Started

Simply open the `index.html` file in your browser.

A lightweight front-end project for practicing and improving web development skills.
